# Personal Portfolio Website

This repository contains my **personal portfolio website**, developed to present my academic background, technical skills, and projects in a structured and professional manner.

I am a **BCA Second Year student at Dezyne Ecole College, Ajmer**, and this project reflects my interest in frontend web development and clean UI design. 
The website is fully responsive and optimized for a consistent user experience across devices.

---

## Project Overview

The portfolio is designed with a **clean, minimal, and modern user interface**, focusing on clarity, usability, and visual balance. 
It serves as a central platform to showcase my profile for **internships, academic evaluations, and entry-level opportunities**.
